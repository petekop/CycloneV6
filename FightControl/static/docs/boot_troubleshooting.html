Boot Troubleshooting

The boot sequence is controlled by the hr_daemon service. If the system fails to boot or you encounter hardware sensor warnings, focus on diagnosing hr_daemon instead of individual red or blue modules.

Check hr_daemon Status

  • sudo systemctl status hr_daemon – view the current status.
  • sudo systemctl restart hr_daemon – restart the service.
  • sudo journalctl -u hr_daemon -f – follow runtime logs for debugging.

Sensor Error Investigation

When hr_daemon reports sensor errors, inspect each strap connection individually:

  • Verify that each strap is connected securely to the hardware.
  • Look for signs of wear, damage, or loose connectors.
  • Swap suspect straps with known-good ones to isolate faulty hardware.

After addressing any strap issues, restart hr_daemon and re-check logs to confirm the problem is resolved.
